## 重复字符串
* 注释
```
n=int(input())
s=list(map(str,input().strip()))
if len(s)%n!=0:
    print(1)
else:
    number=0
    t = len(s) // n
    nums=[[0]*t for i in range(n)]  
    for i in range(n):
        for j in range(t):
            nums[i][j]=s[i*t+j]

    for i in range(t):
        count=0
        s=[]
        for j in range(n):
            s.append(nums[j][i])
        for j in range(n):
            a=s.count(nums[j][i])
            count=max(count,a)
        number+=(abs(n-count))
    print(number)

```
